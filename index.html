About Mouse Tester Pro


Mouse Tester Pro is a browser-based diagnostic and interaction analysis tool designed to evaluate mouse performance, input consistency, and user control. Built with a focus on precision and real-time feedback, it provides both visual and analytical insights into mouse behavior.


Core Features


  • Click Per Second (CPS): Measures click speed in real time.
  • Latency Estimation: Approximates input delay using frame timing.
  • Polling Rate Monitoring: Visualizes mouse report frequency (Hz) based on movement events.
  • Real-Time Graph: Displays latency and polling spikes dynamically.
  • Jitter Detection: Analyzes movement consistency and micro-variation.
  • Drag Test (Slider Mode): Controlled horizontal movement test for precision and stability.
  • Interactive Mouse UI: Visual feedback for all mouse buttons including side buttons.
  • Scroll Wheel Simulation: Directional visual response to scroll input.
  • Hover Highlight Zones: UI reacts to cursor presence for better interaction clarity.
  • Scoped Input System: Testing occurs only within defined zones to ensure accurate measurement.

Technical Notes


  • Polling rate is estimated using event frequency, not raw USB data.
  • Latency is approximated via requestAnimationFrame timing.
  • Side button detection may vary depending on browser and OS limitations.
  • No external libraries are used — fully built with native JavaScript, HTML, and CSS.

Purpose


This project serves as both a diagnostic utility and an experimental platform for analyzing human input behavior, hardware performance, and interaction design.
