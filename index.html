About Mouse Tester Pro


Mouse Tester Pro is a browser-based diagnostic and interaction analysis tool designed to evaluate mouse performance, input consistency, and user control. Built with a focus on precision and real-time feedback, it provides both visual and analytical insights into mouse behavior.


Core Features


  • Click Per Second (CPS): Real-time click speed tracking with per-second refresh.
  • Latency Estimation: Frame-based input delay approximation using high-resolution timing.
  • Polling Rate Monitoring: Measures mouse event frequency (Hz) from movement activity within the test area.
  • Max Polling Detection: Captures peak polling spikes to estimate hardware capability beyond average values.
  • Raw Input Mode: Utilizes pointerrawupdate when supported for higher-fidelity input tracking.
  • Fallback System: Automatically switches to mousemove when raw input is unavailable.
  • Real-Time Graph: Visualizes latency and polling fluctuations including spikes and instability.
  • Jitter Detection: Analyzes inconsistencies in cursor movement to detect instability or noise.
  • Drag Test (Slider Mode): Controlled movement test for evaluating precision, smoothness, and stability.
  • Interactive Mouse UI: Visual feedback for all mouse inputs including left, right, middle, and side buttons.
  • Scroll Wheel Simulation: Direction-based animation reflecting real scroll input behavior.
  • Hover Highlight Zones: Interactive regions respond visually to cursor position.
  • Scoped Input System: All measurements are isolated within defined test zones for consistency.

Technical Notes


  • Polling Rate Limitation: Values are based on browser event delivery and may not reflect true USB polling rate.
  • Raw Input Support: pointerrawupdate is only available in certain browsers (Chromium-based recommended).
  • Latency Measurement: Calculated using performance.now() and frame scheduling techniques.
  • Event-Based Sampling: Data is derived from input events, not direct hardware access.
  • Side Button Detection: May vary depending on browser, OS, and mouse driver support.
  • No External Dependencies: Fully built using native HTML, CSS, and JavaScript for maximum performance.

Purpose


This project serves as both a diagnostic utility and an experimental platform for analyzing human input behavior, hardware performance, and interaction design.
